# عروضي

عروضي هو موقع Static Web App عربي لمتابعة عروض البنوك والمحافظ السعودية في مكان واحد.

يعرض الموقع عروض الكاشباك والخصومات والتقسيط والنقاط، مع فلاتر حسب الفئة، المزود، نوع العرض، شبكة الدفع، والعروض المحفوظة.

## الملفات الرئيسية

- `index.html`: واجهة الموقع الرئيسية.
- `offers.json`: مصدر بيانات العروض.
- `CNAME`: إعداد الدومين المخصص.
- `.nojekyll`: تعطيل معالجة Jekyll في GitHub Pages.
